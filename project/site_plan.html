Anime Fan Site Project Plan

WDD 231 Individual Project

Overview

This document outlines the plan for an individual website project: an Anime Fan Site. The site will be a three-page informational platform dedicated to various aspects of anime. This plan serves as a blueprint and will be updated as the project progresses.

1. Site Name

"AniRealm"

Reason for Selection: The name "AniRealm" combines "Anime" with "Realm," suggesting a vast and immersive world dedicated to anime. It aims to evoke a sense of community and comprehensive content for anime enthusiasts, making it memorable and relevant to the site's purpose.

2. Site Purpose

The primary purpose of AniRealm is to serve as a comprehensive informational hub for anime fans. It will provide engaging content including introductions to various anime genres, detailed reviews and recommendations for series and movies, and spotlights on popular characters and influential animation studios. The site aims to be a go-to resource for discovering new anime, revisiting classics, and deepening appreciation for the art form.

3. Scenarios

Here are two scenarios representing typical questions a site visitor might have:

  • "I'm new to anime and don't know where to start. Can this site help me find a good series based on my interest in fantasy?"
  • "I just finished a popular anime and want to find similar recommendations or learn more about the studio that produced it. Where can I find that information?"

4. Color Schema

The chosen color schema aims for a clean, modern, and engaging look, suitable for an anime-themed site without being overly flashy. The colors are:

  • Primary Accent Color: #4A90E2 (Vibrant Blue)

    Used for: Main headings, navigation links, interactive elements (buttons), and accents to draw attention.

  • Background Color: #F3F4F6 (Light Grey)

    Used for: General page background, providing a clean and subtle backdrop.

  • Body Text Color: #333333 (Dark Grey)

    Used for: Paragraph text and general content, ensuring high readability against the light background.

  • Heading Text Color: #1A202C (Very Dark Grey/Near Black)

    Used for: Section titles and prominent headings, providing strong contrast and visual hierarchy.

5. Typography

The chosen font for the website is 'Inter', providing a modern and legible aesthetic:

  • Font Family: 'Inter', sans-serif

    Used for: All text on the site, including headings, body paragraphs, and navigation. This ensures consistency and a clean, readable user experience.

6. Wireframe (Home Page Layout Sketch)

Below are conceptual sketches for the home page layout for both mobile and desktop views

Mobile View

Description (Mobile): A single-column layout with stacked blocks. The header contains the site logo/name and a menu icon. The hero section features a welcome message and a rotating image. Below are sections for anime genres and featured anime, both designed for vertical scrolling. The footer contains copyright and social media links.

Desktop View

Description (Desktop): A multi-column layout. The header spans full width with logo and navigation. A large hero section follows. Below, a two-column layout features "Anime Genres" on the left (smaller column) and "Featured/Recommended Anime" on the right (larger column), both containing multiple content cards.
